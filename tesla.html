Tesla's modeller

Tesla har lavet mange modeller af biler i deres mange år Deres modeller er model X, model s, model y, model x og cybertruck. Sidstnævnte er dog kun tlgængelig i USA.

Model X

Model X er deres største model og er særligt kendt for deres døre, der åbner opad og ikke udad som almindelige bildøre.

Model Y

Model Y er deres senteste skud fra stammen, derers nyeste model. Det er den mest budget venlige bil.
